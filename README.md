#  Snake Game

Eine Umsetzung des klassischen Snake-Spiels in C# mit Windows Forms.

---

## Über das Projekt

Dieses Projekt ist eine vollständige Implementierung des klassischen Snake-Spiels, entwickelt in C# mithilfe von Windows Forms. Ziel war es, ein funktionierendes Spiel zu entwickeln, das sowohl über die Tastatur als auch über einen Game-Controller gesteuert werden kann.

Die Spiellogik umfasst die Bewegung der Schlange, das Erkennen von Kollisionen mit Wänden und dem eigenen Körper, das zufällige Erzeugen von Äpfeln sowie das dynamische Verlängern des Schwanzes nach dem Fressen. Zusätzlich wurde ein Game-Over-System implementiert, das einen direkten Neustart ermöglicht.

---

## Funktionen

- **Schlangenbewegung** – Die Schlange bewegt sich kontinuierlich und wächst nach jedem gefressenen Apfel
- **Kollisionserkennung** – Berührung mit der Wand oder dem eigenen Körper führt sofort zum Game Over
- **Zufällige Äpfel** – Nach jedem gefressenen Apfel erscheint ein neuer an einer zufälligen Position
- **Punktestand** – Der aktuelle Punktestand wird während des Spiels angezeigt
- **Controller-Unterstützung** – Steuerung wahlweise per Tastatur (Pfeiltasten) oder Game-Controller (`RawGameController`)
- **Neustart** – Nach einem Game Over kann das Spiel direkt neu gestartet werden
